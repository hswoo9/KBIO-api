1. 과도한 메소드 길이

메소드가 100줄을 초과하는 경우가 다수 발견되었습니다.

파일 메소드 라인 수
src/main/java/.../UserController.java processUserRegistration 156
src/main/java/.../BoardService.java updateBoardContent 143

2. 순환 복잡도 초과

순환 복잡도가 10을 초과하는 메소드들이 발견되었습니다.

파일 메소드 복잡도
src/main/java/.../SecurityUtil.java validateUserPermissions 15
src/main/java/.../DataProcessor.java processInputData 12

3. 미사용 임포트

사용되지 않는 임포트 구문이 다수 발견되었습니다.

import java.util.stream.Collectors;  // 미사용
import org.springframework.util.StringUtils;  // 미사용
import java.util.Optional;  // 미사용
